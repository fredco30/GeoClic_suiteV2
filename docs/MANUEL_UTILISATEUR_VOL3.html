Chapitre 9 : Génération de QR Codes

9.1 Principe et utilité des QR codes

Les QR codes (Quick Response codes) sont des codes-barres 2D qui permettent d'accéder instantanément à l'information d'un point en le scannant avec un smartphone.

Cas d'usage des QR codes dans GéoClic Data :
  • Identification sur le terrain : Coller un QR code sur un équipement pour l'identifier rapidement
  • Signalement citoyen : Permettre aux habitants de signaler un problème sur un équipement précis
  • Maintenance : Les agents scannent le QR code pour accéder à l'historique de l'équipement
  • Inventaire : Vérification rapide des équipements lors des tournées
  • Information publique : Afficher des informations sur les arbres remarquables, monuments, etc.

9.2 Générer un QR code pour un point

1 Ouvrir la fiche du point

Dans le module Points ou sur la carte, cliquez sur le point pour ouvrir sa fiche détaillée.

2 Accéder à l'onglet QR Code

Dans la fiche du point, cliquez sur l'onglet "QR Code".

3 Générer le QR code

Cliquez sur Générer QR Code. Le code s'affiche instantanément.

4 Télécharger ou imprimer

Utilisez les boutons pour :

  • Télécharger PNG - Image haute résolution
  • Télécharger PDF - Format imprimable
  • Imprimer - Impression directe

9.3 Générer des QR codes en masse

Pour générer plusieurs QR codes à la fois :

1 Accéder au module QR Codes

Dans le menu latéral, cliquez sur "QR Codes".

2 Sélectionner les points

Plusieurs méthodes de sélection :

  • Par catégorie : Sélectionnez une ou plusieurs catégories
  • Par filtre : Appliquez des critères de filtrage
  • Par sélection manuelle : Cochez les points dans la liste
  • Tout sélectionner : Génère pour tous les points visibles
3 Configurer la génération
  • Taille : Petite (2cm), Moyenne (4cm), Grande (6cm), Personnalisée
  • Format de sortie : PDF (recommandé pour impression) ou ZIP d'images
  • Mise en page : Nombre de QR codes par page
  • Inclure le libellé : Ajouter le nom du point sous le QR code
  • Inclure l'ID : Ajouter l'identifiant unique
4 Générer et télécharger

Cliquez sur Générer X QR Codes. Le fichier est téléchargé.

Astuce impression : Pour les étiquettes adhésives, choisissez le format PDF et configurez la mise en page selon votre papier d'étiquettes (A4 avec 24, 48 ou 65 étiquettes).

9.4 Personnaliser les QR codes

GéoClic Data permet de personnaliser l'apparence des QR codes :

Options de personnalisation

  • Couleur du QR code : Noir par défaut, mais peut être changé (attention à la lisibilité)
  • Couleur de fond : Blanc par défaut
  • Logo central : Ajouter le logo de votre collectivité au centre
  • Bordure : Ajouter un cadre autour du QR code
  • Niveau de correction : L, M, Q ou H (plus le niveau est élevé, plus le QR code résiste aux dégradations)
Attention à la lisibilité :
  • Gardez un contraste élevé entre le QR code et le fond
  • Le logo central ne doit pas dépasser 30% de la surface
  • Testez toujours la lecture du QR code avant impression en masse

9.5 Imprimer les QR codes

Supports d'impression recommandés

Support Durabilité Usage
Étiquettes papier 1-2 ans (intérieur) Mobilier de bureau, archives
Étiquettes plastifiées 3-5 ans Équipements intérieurs
Plaques aluminium 10+ ans Mobilier urbain, extérieur
Gravure laser Permanent Équipements métalliques
Médaillons céramique Permanent Arbres, patrimoine

Conseils d'impression

  • Résolution minimale : 300 dpi pour une bonne qualité
  • Taille minimale : 2cm x 2cm pour une lecture fiable
  • Taille recommandée : 4cm x 4cm pour un confort de lecture
  • Marges : Prévoir une zone blanche autour du QR code

9.6 Scanner un QR code

Le scan d'un QR code GéoClic peut se faire de plusieurs manières :

Avec l'application GéoClic Mobile

  1. Ouvrez l'application GéoClic Mobile
  2. Appuyez sur l'icône de scan QR code
  3. Pointez la caméra vers le QR code
  4. La fiche du point s'ouvre automatiquement

Avec l'appareil photo du smartphone

  1. Ouvrez l'appareil photo de votre smartphone
  2. Pointez vers le QR code (sans prendre de photo)
  3. Un lien apparaît - appuyez dessus
  4. La page du point s'ouvre dans le navigateur

Avec une application de scan tierce

Vous pouvez utiliser n'importe quelle application de scan QR code. Le lien redirigera vers la page publique du point.

9.7 Page publique de point

Chaque point possède une page publique accessible via son QR code. Cette page affiche :

  • Le nom et la catégorie du point
  • Sa localisation sur une carte
  • Les informations publiques configurées
  • Les photos (si configuré)
  • Un formulaire de signalement (optionnel)
Configuration de la page publique :

Dans les paramètres de chaque catégorie, vous pouvez définir quels champs sont visibles sur la page publique. Cela permet de partager certaines informations tout en gardant d'autres confidentielles.

URL de la page publique :

Format : https://votre-domaine.fr/point/{id}

Exemple : https://geoclic.maville.fr/point/ARBRE-001

Chapitre 10 : Exports de Données

10.1 Vue d'ensemble des exports

GéoClic Data permet d'exporter vos données dans différents formats pour les utiliser dans d'autres logiciels ou pour l'archivage.

Format Extension Usage principal Logiciels compatibles
CSV .csv Tableur, base de données Excel, LibreOffice, Access
Excel .xlsx Tableur avec mise en forme Microsoft Excel, LibreOffice
GeoJSON .geojson Données géographiques web QGIS, Leaflet, Mapbox
Shapefile .shp SIG professionnel ArcGIS, QGIS, MapInfo
PDF .pdf Rapport imprimable Tout lecteur PDF
KML/KMZ .kml/.kmz Visualisation carte Google Earth, Maps

10.2 Export au format CSV

Le format CSV (Comma-Separated Values) est le plus universel et compatible avec la plupart des logiciels.

1 Accéder au module Exports

Dans le menu latéral, cliquez sur "Exports".

2 Sélectionner les données
  • Choisissez les catégories à exporter
  • Appliquez des filtres si nécessaire (date, zone, etc.)
  • Prévisualisez le nombre de points concernés
3 Configurer l'export CSV
  • Séparateur : Virgule (,) ou Point-virgule (;) - utilisez ; pour Excel français
  • Encodage : UTF-8 (recommandé) ou Windows-1252
  • Champs à exporter : Sélectionnez les colonnes souhaitées
  • En-têtes : Inclure la ligne de titres
4 Télécharger

Cliquez sur Exporter en CSV. Le fichier est téléchargé.

Pour Excel en français : Utilisez le séparateur point-virgule (;) et l'encodage Windows-1252 pour une ouverture directe sans problème d'accents.

10.3 Export au format Excel

L'export Excel produit un fichier .xlsx avec mise en forme automatique.

Avantages du format Excel :

  • Mise en forme automatique (en-têtes en gras, colonnes ajustées)
  • Plusieurs onglets possibles (un par catégorie)
  • Filtres Excel pré-configurés
  • Formules de synthèse optionnelles

Options spécifiques :

  • Un onglet par catégorie : Sépare les données par type
  • Inclure les photos : Intègre les miniatures dans les cellules
  • Ajouter un onglet statistiques : Synthèse avec graphiques

10.4 Export au format GeoJSON

Le GeoJSON est le format standard pour les données géographiques sur le web.

Caractéristiques :

  • Format texte lisible (JSON)
  • Inclut la géométrie (point, ligne, polygone)
  • Compatible avec tous les outils web de cartographie
  • Système de coordonnées WGS84 (EPSG:4326)

Options :

  • Précision des coordonnées : Nombre de décimales (6 recommandé)
  • Propriétés à inclure : Sélectionnez les champs
  • Fichier unique ou par catégorie

10.5 Export au format Shapefile

Le Shapefile est le format standard des logiciels SIG professionnels (QGIS, ArcGIS).

Particularités du Shapefile :
  • Composé de plusieurs fichiers (.shp, .shx, .dbf, .prj)
  • Les noms de champs sont limités à 10 caractères
  • Ne supporte pas les caractères spéciaux dans les noms de colonnes
  • Export en fichier ZIP contenant tous les fichiers

Options :

  • Système de projection : WGS84 (GPS) ou Lambert 93 (France)
  • Encodage des caractères : UTF-8 ou Windows-1252

10.6 Export PDF et rapports

L'export PDF génère un document formaté prêt à imprimer.

Types de rapports PDF

Type Description Usage
Fiche individuelle Une page par point avec toutes les infos Dossier technique, archive
Liste tabulaire Tableau récapitulatif des points Inventaire, contrôle
Rapport avec carte Carte + liste des points Présentation, réunion
Atlas cartographique Une carte par zone/quartier Navigation terrain
Rapport statistique Graphiques et synthèse Tableau de bord, bilan

Personnalisation du PDF

  • En-tête/Pied de page : Logo, titre, date, numérotation
  • Orientation : Portrait ou Paysage
  • Format : A4, A3, Letter
  • Inclure les photos : Miniatures ou taille réelle

10.7 Planifier des exports automatiques

GéoClic Data permet de planifier des exports récurrents pour automatiser les sauvegardes ou les envois de données.

1 Créer une tâche planifiée

Dans le module Exports, cliquez sur Planifier un export.

2 Configurer la tâche
  • Nom : Identifiant de la tâche
  • Données : Catégories et filtres
  • Format : CSV, Excel, GeoJSON...
  • Fréquence : Quotidien, Hebdomadaire, Mensuel
  • Heure d'exécution : Moment de la journée
3 Définir la destination
  • Email : Envoi en pièce jointe à une liste de destinataires
  • Stockage : Sauvegarde sur le serveur (historique)
  • FTP/SFTP : Dépôt sur un serveur distant
4 Activer la planification

Activez la tâche. Elle s'exécutera automatiquement selon la planification définie.

Exemple : Export automatique hebdomadaire
  • Nom : "Backup hebdo patrimoine arboré"
  • Données : Catégorie "Arbres" - Tous
  • Format : Excel
  • Fréquence : Tous les lundis à 6h00
  • Destination : Email à service.espaces-verts@maville.fr

Chapitre 11 : Imports de Données

11.1 Formats de fichiers acceptés

GéoClic Data accepte plusieurs formats de fichiers pour l'import de données :

Format Extension Notes
CSV .csv Séparateur virgule ou point-virgule
Excel .xlsx, .xls Premier onglet utilisé par défaut
GeoJSON .geojson, .json Coordonnées automatiquement extraites
Shapefile .zip (contenant .shp) Archive ZIP obligatoire
KML/KMZ .kml, .kmz Export Google Earth/Maps

11.2 Préparer son fichier d'import

Une bonne préparation du fichier garantit un import sans erreur.

Structure recommandée (CSV/Excel)

Colonne Obligatoire Format Exemple
latitude Oui Nombre décimal 48.856600
longitude Oui Nombre décimal 2.352200
categorie Oui Code lexique ARBRE
identifiant Non* Texte unique ARB-001
[champs dynamiques] Non Selon le type Tilleul, 15, 2020-05-15

* Si non fourni, un identifiant sera généré automatiquement

Règles de préparation

  • Première ligne : Noms des colonnes (en-têtes)
  • Coordonnées : Format décimal, point comme séparateur décimal
  • Dates : Format AAAA-MM-JJ (ISO) ou JJ/MM/AAAA
  • Valeurs vides : Laisser la cellule vide, pas de "NULL" ou "N/A"
  • Catégorie : Utiliser le CODE du lexique (pas le libellé)
Erreurs courantes à éviter :
  • Coordonnées inversées (longitude/latitude)
  • Virgule comme séparateur décimal dans les coordonnées
  • Code de catégorie inexistant ou mal orthographié
  • Fichier Excel avec des formules au lieu de valeurs
  • Lignes vides au milieu du fichier

11.3 Procédure d'import pas à pas

1 Accéder au module Imports

Dans le menu latéral, cliquez sur "Imports".

2 Charger le fichier

Cliquez sur Sélectionner un fichier ou glissez-déposez votre fichier dans la zone.

3 Analyser le fichier

GéoClic analyse automatiquement le fichier et détecte :

  • Le nombre de lignes
  • Les colonnes présentes
  • Le format des données
  • Les erreurs éventuelles
4 Mapper les colonnes

Associez chaque colonne du fichier à un champ GéoClic (voir section suivante).

5 Prévisualiser

Vérifiez les premières lignes. Corrigez les erreurs si nécessaire.

6 Lancer l'import

Cliquez sur Importer X points.

7 Vérifier le résultat

Un rapport d'import s'affiche avec le nombre de points créés, mis à jour, et les éventuelles erreurs.

11.4 Mapping des colonnes

Le mapping (ou mappage) permet d'associer les colonnes de votre fichier aux champs de GéoClic Data.

Mapping automatique

GéoClic tente de détecter automatiquement les correspondances basées sur :

  • Le nom de la colonne (latitude, long, cat, etc.)
  • Le format des données
  • Les imports précédents

Mapping manuel

Pour chaque colonne non reconnue :

  1. Cliquez sur le menu déroulant à côté du nom de colonne
  2. Sélectionnez le champ GéoClic correspondant
  3. Options disponibles :
    • Latitude / Longitude : Coordonnées GPS
    • Catégorie : Code du lexique
    • Identifiant : ID unique du point
    • [Nom du champ] : Champs dynamiques
    • Ignorer : Ne pas importer cette colonne
Sauvegarder le mapping : Vous pouvez enregistrer votre configuration de mapping pour la réutiliser lors de prochains imports similaires.

11.5 Gestion des erreurs d'import

Lors de l'analyse et de l'import, des erreurs peuvent survenir :

Erreur Cause Solution
Coordonnées invalides Format incorrect ou hors limites Vérifier le format décimal et les valeurs
Catégorie inconnue Code inexistant dans le lexique Créer la catégorie ou corriger le code
Champ obligatoire vide Valeur manquante Compléter le fichier
Type de donnée incorrect Texte au lieu de nombre, etc. Corriger le format des valeurs
Doublon d'identifiant ID déjà existant Modifier l'ID ou faire une mise à jour

Mode de gestion des erreurs

  • Ignorer les erreurs : Importe les lignes valides, ignore les autres
  • Arrêter à la première erreur : Stoppe l'import pour correction
  • Rapport détaillé : Génère un fichier des lignes en erreur

11.6 Import de mise à jour

Pour mettre à jour des points existants plutôt que d'en créer de nouveaux :

  1. Incluez la colonne "identifiant" avec les ID existants
  2. Dans les options d'import, choisissez "Mettre à jour si existant"
  3. Sélectionnez les champs à mettre à jour
Modes de mise à jour :
  • Remplacer : Écrase toutes les valeurs avec celles du fichier
  • Compléter : Met à jour uniquement les champs non vides du fichier
  • Fusionner : Complète les champs vides du point avec le fichier

Chapitre 12 : Gestion des Projets

12.1 Concept de projet

Les projets permettent de regrouper des points pour une opération spécifique, indépendamment de leur catégorie.

Exemples de projets :
  • Campagne d'élagage 2026 : Regroupe les arbres à élaguer cette année
  • Renouvellement éclairage LED : Liste des lampadaires à remplacer
  • Inventaire patrimoine : Points à vérifier lors de l'inventaire
  • Travaux rue de la Liberté : Tous les équipements impactés par les travaux

12.2 Créer un nouveau projet

1 Accéder au module Projets

Dans le menu latéral, cliquez sur "Projets".

2 Créer un projet

Cliquez sur + Nouveau projet.

3 Remplir les informations
  • Nom : Intitulé du projet
  • Description : Détails sur l'objectif
  • Date de début : Début prévu
  • Date de fin : Échéance
  • Responsable : Utilisateur en charge
  • Statut : Planifié, En cours, Terminé, Annulé
4 Enregistrer

Cliquez sur Créer le projet.

12.3 Affecter des points à un projet

Plusieurs méthodes pour ajouter des points à un projet :

Méthode 1 : Depuis la fiche point

  1. Ouvrez la fiche d'un point
  2. Cliquez sur l'onglet "Projets"
  3. Cliquez sur "Ajouter à un projet"
  4. Sélectionnez le projet

Méthode 2 : Depuis la liste de points

  1. Dans le module Points, cochez les points souhaités
  2. Cliquez sur "Actions" → "Ajouter au projet"
  3. Sélectionnez le projet

Méthode 3 : Depuis le projet

  1. Ouvrez le projet
  2. Cliquez sur "Ajouter des points"
  3. Recherchez et sélectionnez les points
  4. Ou utilisez les filtres pour ajouter en masse

12.4 Suivre l'avancement

Pour chaque point d'un projet, vous pouvez définir un statut d'avancement :

  • À faire : Pas encore traité
  • En cours : Traitement en cours
  • Terminé : Action réalisée
  • Annulé : Non pertinent finalement

Le tableau de bord du projet affiche :

  • Nombre total de points
  • Répartition par statut (graphique)
  • Pourcentage d'avancement
  • Points en retard (selon planning)

12.5 Exporter un projet

Depuis la page du projet :

  1. Cliquez sur Exporter
  2. Choisissez le format (Excel, PDF, CSV)
  3. Sélectionnez les informations à inclure
  4. Téléchargez le fichier

L'export inclut :

  • Liste des points avec leurs attributs
  • Statut d'avancement de chaque point
  • Statistiques globales du projet
  • Carte des points (format PDF)

Chapitre 13 : Administration

Accès réservé : Les fonctions d'administration sont réservées aux utilisateurs ayant le rôle Administrateur. Si vous n'avez pas accès à ces fonctions, contactez votre administrateur système.

13.1 Accéder aux fonctions d'administration

Les fonctions d'administration sont accessibles via le menu latéral :

  • Utilisateurs : Gestion des comptes utilisateurs
  • Paramètres : Configuration générale de l'application

13.2 Gestion des utilisateurs

Créer un nouvel utilisateur

1 Accéder au module Utilisateurs

Dans le menu latéral, cliquez sur "Utilisateurs".

2 Ajouter un utilisateur

Cliquez sur + Nouvel utilisateur.

3 Remplir les informations
  • Nom : Nom complet de l'utilisateur
  • Email : Adresse email (servira d'identifiant)
  • Mot de passe : Mot de passe initial (l'utilisateur pourra le changer)
  • Rôle : Niveau de permissions (voir section suivante)
  • Service : Département ou équipe (optionnel)
  • Actif : Cochez pour activer le compte
4 Enregistrer

Cliquez sur Créer l'utilisateur.

Modifier un utilisateur

  1. Cliquez sur l'utilisateur dans la liste
  2. Modifiez les informations souhaitées
  3. Enregistrez les modifications

Désactiver un compte

Plutôt que de supprimer un utilisateur (ce qui peut poser des problèmes de traçabilité), désactivez le compte :

  1. Ouvrez la fiche de l'utilisateur
  2. Décochez "Actif"
  3. Enregistrez

L'utilisateur ne pourra plus se connecter, mais son historique d'actions reste visible.

Réinitialiser un mot de passe

  1. Ouvrez la fiche de l'utilisateur
  2. Cliquez sur "Réinitialiser le mot de passe"
  3. Un mot de passe temporaire est généré
  4. Communiquez-le à l'utilisateur
  5. L'utilisateur devra le changer à sa prochaine connexion

13.3 Rôles et permissions

GéoClic Data propose 4 rôles prédéfinis :

Rôle Description Permissions principales
Administrateur Accès complet Toutes les fonctions + gestion utilisateurs
Gestionnaire Gestion des données CRUD complet + exports + projets
Utilisateur Saisie de données Créer/modifier ses points + consultation
Lecteur Consultation seule Voir les données + exports limités

Détail des permissions

Points
  • Admin/Gestionnaire : CRUD tous
  • Utilisateur : CRUD ses points
  • Lecteur : Lecture seule
Lexique
  • Admin : CRUD complet
  • Gestionnaire : Lecture + suggestion
  • Autres : Lecture seule
Exports
  • Admin/Gestionnaire : Tous formats
  • Utilisateur : CSV/Excel limité
  • Lecteur : PDF rapport seul
Projets
  • Admin/Gestionnaire : CRUD projets
  • Utilisateur : Voir + mettre à jour statut
  • Lecteur : Lecture seule

13.4 Paramètres généraux

Le module Paramètres permet de configurer le comportement global de l'application.

Paramètres de l'organisation

  • Nom de l'organisation : Affiché dans l'interface et les exports
  • Logo : Utilisé dans les PDF et QR codes
  • Coordonnées de contact
  • Zone géographique par défaut : Centre de la carte au démarrage

Paramètres de la carte

  • Fond de carte par défaut
  • Niveau de zoom initial
  • Limites de la zone (bounding box)
  • Clustering activé/désactivé

Paramètres de sécurité

  • Durée de session : Délai avant déconnexion automatique
  • Politique de mot de passe : Longueur minimale, complexité
  • Double authentification : Activer/désactiver 2FA

13.5 Sauvegarde et restauration

Sauvegarde manuelle

  1. Allez dans Paramètres → Sauvegarde
  2. Cliquez sur "Créer une sauvegarde"
  3. Attendez la génération
  4. Téléchargez le fichier de sauvegarde

La sauvegarde inclut :

  • Toutes les données (points, lexique, utilisateurs)
  • Les fichiers attachés (photos, documents)
  • Les paramètres de configuration

Sauvegarde automatique

Configurez des sauvegardes automatiques :

  • Fréquence : Quotidienne, Hebdomadaire
  • Heure : Moment de la sauvegarde (nuit recommandée)
  • Rétention : Nombre de sauvegardes conservées
  • Destination : Locale ou distante (FTP/S3)

Restauration

Attention : La restauration remplace TOUTES les données actuelles par celles de la sauvegarde. Cette action est irréversible. Faites une sauvegarde avant de restaurer.
  1. Allez dans Paramètres → Sauvegarde
  2. Cliquez sur "Restaurer une sauvegarde"
  3. Sélectionnez le fichier de sauvegarde
  4. Confirmez la restauration
  5. Attendez la fin du processus

13.6 Journaux d'activité

GéoClic Data enregistre toutes les actions importantes pour la traçabilité.

Types d'événements enregistrés

  • Connexions et déconnexions
  • Création, modification, suppression de points
  • Modifications du lexique
  • Exports de données
  • Imports de données
  • Actions d'administration

Consulter les journaux

  1. Allez dans Paramètres → Journaux
  2. Utilisez les filtres :
    • Par période (date de début/fin)
    • Par utilisateur
    • Par type d'action
    • Par élément concerné
  3. Consultez la liste des événements

Exporter les journaux

Pour l'audit ou l'archivage, exportez les journaux en CSV ou PDF.

Conservation des journaux : Les journaux sont conservés selon la politique de rétention configurée (par défaut 1 an). Pour des raisons légales ou de conformité, vous pouvez ajuster cette durée dans les paramètres.
